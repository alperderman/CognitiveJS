Repeaters' purpose in CognitiveJS is to iterate through given array (similar to ngRepeat in AngularJS). Repeaters can be defined with cog-repeat attribute.

Repeaters are built on top of templating functionality, so every repeater has to include a template ID. If you haven't load the template with cog-set attribute, then it will automatically load the current elements inside the repeater as a template.

Inside the repeaters, _index token keyword can be used to get the index number of each iteration. Multi-dimensional repeaters inside one another can be defined. Additionally repeater definitions inside normal template definitions is also allowed (though it makes things a bit complicated since you have to work on with multiple IDs and aliases).

Root element of the repeater can be accessed via cog.id and cog.ids objects, just like a template.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <ul cog-repeat="list_template;item;fruits">
                <li>{{item._index}} - {{item}}</li>
            </ul>

            <script src="../dist/cog.js"></script>
            <script>
                cog.data.fruits = ["apple", "banana", "berry"];
                cog.render();
            </script>

        </body>
        </html>
