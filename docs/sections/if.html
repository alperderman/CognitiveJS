Conditional rendering is the way of showing elements or changing properties of elements based on set conditions. In order to display or hide element based on condition, cog-if attribute can be used. Conditions on props, events or lives can be set via an array with the key "if" inside their attribute.

Operation of "if array" inside props is simple. First value of "if array" is the condition itself, second value is if the condition returns true, third parameter is not required but it is for if it returns false.

Multiple "if arrays" can be declared inside one another to achieve else if functionality.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <h1 cog-prop="if:[{{bool1}}, {style:{color:'red'}}, {if:[{{bool2}}, {style:{color:'blue'}}, {style:{color:'green'}}]}]">Title!</h1>

            <button cog-event="if:[!{{bool2}}, {click:'clickButton()', mouseover:'hoverButton()', _prevent:true}]">Send</button>
        
            <input cog-live="if:[{{bool1}}, {live:'username', event:'input'}]" cog-prop="context:{value:{{username}}}" type="text">
        
            <p cog-if="{{bool1}}">Lorem ipsum</p>
            <p cog-if="{{bool2}}">Dolor sit amet</p>

            <script src="../dist/cog.js"></script>
            <script>
                cog.data.bool1 = false;
                cog.data.bool2 = true;
                cog.data.username = "";

                function clickButton() {
                    console.log("click");
                }
                function hoverButton() {
                    console.log("hover");
                }

                cog.render();
            </script>

        </body>
        </html>
