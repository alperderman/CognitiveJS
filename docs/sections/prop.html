Properties allows us to dynamically modify the elements as we please. Styles, classes, attributes and node properties (called context for the sake of differentiation) can be declared using cog-prop attribute on elements.

Multiple cog-prop attributes can be defined in a single element, for example cog-prop1, cog-prop2, cog-propXY. As long as the name of the attribute starts with "cog-prop", it will be treated as a property attribute.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <h1 cog-prop="style:{color:'#f00'}">TITLE</h1>

            <p 
            cog-prop1="style:{backgroundColor:{{bgcolor}}, color:'#fff'}, class:'class1 '+{{class}}" 
            cog-prop2="class:'class3'">Lorem ipsum dolor sit amet.</p>

            <a cog-prop="attr:{href:'http://google.com'}">Link</a>

            <input cog-prop="context:{value:'input value'}" type="text">

            <script src="../dist/cog.js"></script>
            <script>
                cog.data.bgcolor = "#00f";
                cog.data.class = "class2";
                cog.render();
            </script>

        </body>
        </html>
