In CognitiveJS event are not defined with addEventListener() function or any equivalent, instead they are defined with static cog-event attribute and they will stay after render is completed.

Setting _prevent option to true inside event attribute will prevent the event from bubbling up, so that multiple amount of same event types can be declared inside elements inside one another without triggering the parent element.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <button cog-event="click:'clickButton()', mouseover:'hoverButton()', _prevent:true">Send</button>

            <script src="../dist/cog.js"></script>
            <script>
                function clickButton() {
                    console.log("click");
                }
                function hoverButton() {
                    console.log("hover");
                }
                cog.render();
            </script>

        </body>
        </html>
