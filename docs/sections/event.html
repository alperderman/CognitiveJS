{{section.title}}

In CognitiveJS events are defined with cog-event attribute and they will stay after render is completed. Tokens and conditional rendering is allowed inside the attribute to make dynamic changes possible.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <button cog-event="click:'clickButton()', mouseover:'hoverButton()'">Send</button>

            <script src="../dist/cog.js"></script>
            <script>
                function clickButton() {
                    console.log("click");
                }
                function hoverButton() {
                    console.log("hover");
                }
                cog.render();
            </script>

        </body>
        </html>

{{section.sub.0.title}}

Two-way binding is achieved with a slight modification to the event attribute. Instead of defining a function, defining an object with set as the key and token as the value, will change the value of the token with the value of the input every time the specified event is fired. Combining this with context prop, will give you the ability to change the value of the input programatically, which completes the two-way binding circle.

Default data (which is the "value" property of the input) to be set as the token value can be change via data property inside the object. For a better understanding about this technique, look at the "Form Example" in Examples section.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <input cog-prop="context:{value:{{username}}}" 
            cog-event="input:{set:'username'}"
            type="text">
            
            <button cog-event="click:changeUsername">Change Input</button>

            <p>Username: {{username}}</p>

            <script src="../dist/cog.js"></script>
            <script>
                cog.data.username = "";

                function changeUsername() {
                    cog.data.username = "my_username1995";
                }

                cog.render();
            </script>

        </body>
        </html>
