Introduction

CognitiveJS (previously called DCG) is a front-end JavaScript framework/library for rendering dynamically declared contents and templates on DOM. Compared to the other frameworks, Cognitive aims to write minimal amount code as possible.

Cognitive broadly inspired by Knockout. As opposed to Knockout however, Cognitive doesn't have a design pattern and it can be used as a pre-processing framework with certain modifications.

Download from GitHub

After downloading the CognitiveJS from GitHub, There are two main ways of using it in your project:

First way is the traditional way, which is calling a function that initiates the CognitiveJS directly on the current document.

    
        <html>
        <head>
            <title>CognitiveJS</title>
        </head>
        <body>

            <p data-prop="text:'%hello%'"></p>

            <script src="cog.js"></script>
            <script>
                cog.data = {
                    hello:"Hello World"
                };
                cog.bindAll();
            </script>

        </body>
        </html>

Second way is the pre-processing way that renders a document with the specified path. This is the intended way of using Cognitive and it has couple of benefits such as managing contents and layouts seperately with ease. Explained thoroughly on the section.

            
                <html>
                <head>
                    <title>Main</title>
                </head>
                <body>
    
                    <div data-set="text:hello">Hello World</div>
    
                    <script src="cog.js"></script>
                    <script>
                        cog.render("path/to/layout.html");
                    </script>
    
                </body>
                </html>
            
        
            
                <html>
                <head>
                    <title>Layout</title>
                    <link rel="stylesheet" type="text/css" href="your_styles.css" />
                </head>
                <body>
    
                    <p data-prop="text:'%hello%'"></p>
    
                    <script src="your_scripts.js"></script>

                </body>
                </html>
